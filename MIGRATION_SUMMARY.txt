COMPONENTS USING AUTHENTICATED USER
================================================================================

All page components now use authenticated user:

1. Profile.js
   - Gets user ID: useSelector(selectUserProfile)?.id
   - Updates user profile: dispatch(updateUser(data))
   
2. Discovery.js
   - Gets user ID: userProfile?.id
   - Loads matching users for this user
   
3. Preferences.js
   - Gets user ID: userProfile?.id
   - Saves preferences for this user
   
4. Questionnaire.js
   - Gets user ID: userProfile?.id
   - Saves questionnaire for this user
   
5. Matches.js
   - Gets user ID: userProfile?.id
   - Loads matches for this user
   
6. Layout.js
   - Displays user name: user?.firstName
   - Handles logout: clears auth state
