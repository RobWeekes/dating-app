## SCORING ALGORITHM

### Step 1: Calculate Dimension Scores

**Personality Score** (Q1-Q10):

```
Average responses across 5 Big Five traits (after reverse scoring)
Result: 1-5 scale (5 = high on that trait)
```

**Values Score** (Q11-Q18):

```
Extract top 2 values from Q11-Q12
Calculate alignment with partner's top values
Result: 0-100 score based on overlap
```

**Family Score** (Q19-Q23):

```
Check for critical mismatches:
- Q19 & Q22 must align (children desire + commitment level)
- Q23 must match (monogamy preference)
Critical mismatch = automatic incompatibility flag
Result: Flagged or 0-100 score
```

**Financial Score** (Q24-Q29):

```
Calculate average difference across financial dimensions
Large difference on Q24 or Q25 = major penalty
Result: 0-100 score
```

**Lifestyle Score** (Q30-Q37):

```
Calculate compatibility on sleep schedule (Q30-Q31)
Social frequency alignment (Q33)
Cleanliness/organization mismatch (Q35)
Result: 0-100 score
```

**Work-Life Score** (Q38-Q42):

```
Calculate career priority alignment (Q38-Q39)
Work hours expectation match (Q39)
Result: 0-100 score
```

**Health Score** (Q43-Q46):

```
Calculate health consciousness alignment
Check for substance use alignment (Q44-Q45)
Result: 0-100 score
```

**Physical Score** (Q47-Q50):

```
Check age compatibility (Q47)
Calculate physical preference alignment
Result: 0-100 score
```

### Step 2: Apply Weighting

```
Overall Compatibility Score = (
  Personality × 0.25 +
  Values × 0.30 +
  Family × 0.25 +
  Financial × 0.20
)

Additional Adjustments:
- Lifestyle compatibility (±5% adjustment)
- Work-Life balance (±5% adjustment)
- Health/substance use (±2.5% adjustment)
- Physical preferences (±2.5% adjustment)
```

### Step 3: Critical Incompatibility Checks

```
IF Q19 & Q22 mismatch (one wants kids, other doesn't):
  → Flag: "Family Planning Incompatibility"
  → Score penalty: -30%

IF Q23 mismatch (monogamy preference differs):
  → Flag: "Relationship Structure Incompatibility"
  → Score penalty: -30%

IF Q24 extreme difference (Saver=1 vs. Spender=10):
  → Flag: "Financial Attitude Conflict"
  → Score penalty: -15%

IF Q35 extreme difference (Fastidious=1 vs. Casual=10):
  → Flag: "Household Standards Conflict"
  → Score penalty: -10%
```

### Step 4: Match Rating

```
Score Ranges:
- 85-100: ⭐⭐⭐⭐⭐ Excellent Match
- 70-84:  ⭐⭐⭐⭐ Good Match
- 50-69:  ⭐⭐⭐ Potential Match
- 35-49:  ⭐⭐ Challenging Match
- 0-34:   ⭐ Low Compatibility
```

### Database Schema (Sequelize Models)

// Answer.js
const Answer = sequelize.define("Answer", {
  questionId: {
    type: DataTypes.INTEGER,
    allowNull: false,
  },
  userId: {
    type: DataTypes.INTEGER,
    allowNull: false,
  },
  answerValue: {
    type: DataTypes.STRING,
    allowNull: false,
  },
  answerType: {
    type: DataTypes.ENUM("likert", "choice", "ranking", "range"),
    allowNull: false,
  },
});

// Question.js
const Question = sequelize.define("Question", {
  questionNumber: {
    type: DataTypes.INTEGER,
    allowNull: false,
  },
  text: {
    type: DataTypes.STRING,
    allowNull: false,
  },
  dimension: {
    type: DataTypes.ENUM(
      "personality",
      "values",
      "family",
      "financial",
      "lifestyle",
      "workLife",
      "health",
      "physical",
    ),
    allowNull: false,
  },
  type: {
    type: DataTypes.ENUM("likert", "choice", "ranking", "range"),
    allowNull: false,
  },
  options: {
    type: DataTypes.JSON,
    allowNull: true,
  },
  conditional: {
    type: DataTypes.JSON,
    allowNull: true, // e.g., { dependsOn: 'Q19', value: ['yes', 'probably'] }
  },
  isCritical: {
    type: DataTypes.BOOLEAN,
    defaultValue: false,
  },
});

// CompatibilityScore.js
const CompatibilityScore = sequelize.define("CompatibilityScore", {
  userId1: {
    type: DataTypes.INTEGER,
    allowNull: false,
  },
  userId2: {
    type: DataTypes.INTEGER,
    allowNull: false,
  },
  personalityScore: {
    type: DataTypes.FLOAT,
  },
  valuesScore: {
    type: DataTypes.FLOAT,
  },
  familyScore: {
    type: DataTypes.FLOAT,
  },
  financialScore: {
    type: DataTypes.FLOAT,
  },
  lifestyleScore: {
    type: DataTypes.FLOAT,
  },
  workLifeScore: {
    type: DataTypes.FLOAT,
  },
  healthScore: {
    type: DataTypes.FLOAT,
  },
  physicalScore: {
    type: DataTypes.FLOAT,
  },
  overallCompatibilityScore: {
    type: DataTypes.FLOAT,
  },
  matchRating: {
    type: DataTypes.INTEGER, // 1-5 stars
  },
  redFlags: {
    type: DataTypes.JSON,
    allowNull: true,
  },
  calculatedAt: {
    type: DataTypes.DATE,
    defaultValue: DataTypes.NOW,
  },
});
